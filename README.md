Manhole Predictor

A Full Stack AI-based Manhole Detection and Prediction System developed using Python, Flask, MySQL, HTML, CSS, JavaScript, OpenCV, and Deep Learning techniques.

Features

User Module

User Registration
User Login
Upload Road Images
View Prediction Results
Map Location Display
View Detection Status

Admin Module

Admin Login
Manage Users
Manage Uploaded Images
View Prediction Reports
Monitor Detection Results

AI Features

Manhole Detection using Deep Learning
Image Processing with OpenCV
Road Safety Monitoring
Prediction Accuracy Display

Technologies Used

Python
Flask
MySQL
HTML
CSS
JavaScript
OpenCV
Deep Learning
YOLO

**How to Run**

**Clone the Repository**

git clone https://github.com/Divakar0805/manhole-predictor.git

Open Project Folder

cd manhole-predictor

Install Required Packages

pip install -r requirements.txt

Configure MySQL Database

Create Database in MySQL
Update Database Username and Password in project

Run Flask Application

python main.py

Open Browser

http://localhost:5000
